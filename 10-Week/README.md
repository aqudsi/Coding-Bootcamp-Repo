## Introduction to MongoDB and ReactJS

### Overview

In this week, we will be introducing students to the NoSQL database with MongoDB and Mongoose. We'll also be showing students how to scrape websites with cheerio, and how to place their scraped data in a MongoDB database. We will also introduce students to React, a declarative, efficient, and flexible JavaScript library for building user interfaces. We'll go over key features of the framework: JSX, components, states, props, lifecycle events and so on. By the end of this unit, students should understand how React lets web devs build quicker and less resource-hungry sites than mere HTML and jQuery.
