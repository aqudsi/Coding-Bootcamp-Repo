## Intro to JavaScript / Intro to jQuery

### Overview
This week we will review JavaScript & jQuery fundamentals, as well as touch on timing events, debugger, and sample interview questions.

Also this week, we will be introducing the concept of APIs and the use of AJAX queries to retrieve and parse data in JSON formats. Throughout the week we'll be specifically working with the Giphy, OMDb, and New York Times APIs.
