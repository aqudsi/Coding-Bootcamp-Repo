## Intro to Constructors and Callbacks & Intro to MySQL

### Overview
In this week, we will be diving back into some more object-oriented programming by introducing constructors and callback functions. Also, we are starting working with databases by introducing MySQL and SQL queries before connecting them to Node and user-input.
